## 🚀 How to Use

1. Open `INPHO_Statistics_Problems.html` in any modern browser (Chrome, Firefox, Edge).
2. Read each problem and attempt all parts independently.
3. Click **"Reveal Full Solution"** to check your work.
4. Solutions are hidden by default — no peeking until you've tried!

> 💡 To print with solutions visible, use **Ctrl+P** — the print stylesheet automatically expands all solution blocks.

## 🎯 Problem Style

Each problem is designed in the style of **INPHO** (Indian National Physics Olympiad) and **Jinhui Wang** problem sets:

- Multi-part questions that build on each other
- Emphasis on **conceptual reasoning** over rote computation
- Algebraic **proofs** and rigorous justifications required
- Real-world contexts (finance, climate, manufacturing, medicine)
- **Counterexamples** and conceptual traps embedded throughout
